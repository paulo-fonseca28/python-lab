# Python Course – Microsoft

## Repository Overview

This directory contains exercises and small projects developed during the “Microsoft Python Development” course.  
The goal is to track my progress in Python and keep a well‑organized lab for experiments and practice.

## Technologies and Tools

- Language: Python 3  
- Development environment: VS Code + WSL (Ubuntu)  
- Version control: Git and GitHub  

## How to Run the Examples

1. Clone this repository:  

```bash
git clone https://github.com/paulo-fonseca28/python-lab.git
```

2. Navigate to the course folde and run any example script with:  

```bash
cd python-lab/python_course_microsoft
python3 (or python) filename.py
```

If a virtual environment or `requirements.txt` is added in the future, the setup instructions will be documented here.
